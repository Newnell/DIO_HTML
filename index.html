Como tudo começou?

Voltar

Bem antes do desenvolvimento de computadores eletrônicos, a espécie humana já pensava em cálculos e solução de problemas. Há 3200 anos fazia-se uso do ábaco, um artefato artesanal utilizado para realização de cálculos matemáticos. Na década de 1940, através do desenvolvimento industrial, os computadores surgiram e passaram por 5 gerações até chegar nas máquinas atuais. Válvulas, transistores, circuitos integrados, microprocessadores e computadores quânticos resume a saga do que hoje parece um órgão do corpo humano.

Internet

Voltar

A intrenet surgiu durante a guerra fria, numa reação americana aos projetos russos em 1957 chamada de projeto DARPA. Nessa época Joseph Carl, psicólogo e cientista da computação americano idealiza uma rede entre computadores onde todos poderiam se comunicar. Surgiram então os sistemas de comunicação em pacotes, os protocolos TCP/IP, a world wide web e a evolução continua.

Aplicações web

Voltar

Nesse contexto existem as aplicações web que são soluções desenvolvidas para realizar suas tarefas diretamente na internet sem a necessidade da instalação no dispositivo. No entanto, aplicações como youtube e instagram funcionam como soluções web ou não.

Linguagens de programação

Voltar

De uma forma geral, as linguagens de programação são conjuntos de instruções com escrita e sentidos específicos utilizados para escrever programas de computador. Existem várias linguagens, algumas mais específicas para desenvolvimento de soluções para o lado do servidor, como javascript e C# e outras mais específicas para o lado do cliente como por exemplo o javascrip.

HTML

Voltar

O HTML por sua vez não é uma linguagem de programação. Do inglês HyperText Markup Language ou linguagem de marcação de texto, o HTML é um arquivo escrito com marcações que são interpretadas pelos navegadores de internet e retornam páginas de internet com configurações específicas, tudo isso de acordo com as marcações ou tags utilizadas na escrita da mesma .

Tags

Voltar

As tags são as marcações que caracterizam um arquivo em HTML. Existem inúmeras tags, cada uma com uma função específica no arquivo, entre essas funções por exemplo está a que define títulos e subtítulos em vários níveis e subníveis (tags h1, h2...h6) como o deste ítem, parágrafos como este (tag p), configurações de negrito ou itálico e links. As tags são abertas com o sinal "<", seguido do nome da tag e o sinal de ">" e fechadas com o sinal de "<", seguido de uma "/" , o nome da tag e o sinal de ">". A tag blockquote é utilizada em textos para fazer citações, usando essa tag o texto ficará mais centralizado e em destaque relacionado ao texto principal.

Atributos das tags

Voltar

Os atributos são propriedades que as tags podem ter, como cor, tipo de entrada, estilo. Elas podem ser genéricas ou específicas para determinada tag. Uma propriedade muito importante é a "id" do elemento, este atributo identifica o elemento dentro do HTML para que o mesmo possa ser manipulado por uma linguagem de programaçao por exemplo.

Listas no HTML

Voltar

As listas no HTML podem ser ordenadas e não ordenadas. Para implementá-las faz-se o uso das tags ol e ul respectivamento. Dentro destas tags aão utilizadas outra tag que representa um item de da lista (tag li).
